# Understanding the repo
Take a look at `./IDEA_OVERVIEW.md` - this contains the original purpose behind this repository
You can also get an idea of what has been done in the past, and why, to get this repo to the state it's currently in, by visiting both the `./agentWorkHistory` folder (which contains the actual work that's been done), and `./projectTasks` folder (which contains steps that have been completed, and steps still underway).

Steps in the `./projectTasks` folder which are still under way should not be seen as your current task unless explicitly called out in the user's request.
